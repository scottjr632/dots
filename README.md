# Dotfiles

To install, clone this reposity in to your home directory.

```bash
git clone https://github.com/scottjr632/dotfiles.git ~/.dotfiles
```

You then need to have [stow](https://www.gnu.org/software/stow/) installed

After that you can run

```bash
cd ~/.dotfiles
stow .
```

this will create symlinks to your dotfiles in your home directory
